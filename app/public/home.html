CODE - BUDDY - FINDER

Use this web application to help find a coding buddy.

  • Coding with a partner is a common practice, so we want to help you find the RIGHT person to tackle a project with.

  • Our goal is to match you up with someone that has strengths complementary to your own. You like design? We'll find you a backend guru! Your weaknesses rounded out by someone else's strengths. That's the power of a "Code Buddy"!

  • Tell us a bit about yourself, and we'll find you a friend that would complement you well as you tackle coding projects together!
